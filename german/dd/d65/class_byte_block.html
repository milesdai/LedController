Ausführliche Beschreibung

Ein Block an bytes, der alle Daten in einem Segment darstellt.

Es ist einfach eine ByteRow mit Länge 8, trotzdem nützlich.

Beispiele
LedControllerDemoCounting.ino, LedControllerDemoHwSPI.ino, LedControllerDemoRocket.ino und LedControllerDemoRocketMulti.ino.
