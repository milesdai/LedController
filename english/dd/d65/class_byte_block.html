Detailed Description

A block of bytes used to represent all of the data in a Segment.

It is simply a ByteRow with size 8 but useful regardless.

Examples
Led-matrix-counting.ino, Led-matrix-message.ino, Led-matrix-rocket-hwSPI.ino, Led-matrix-rocket-multi.ino, and Led-matrix-rocket.ino.
